## Pin assignment
### LED
|Pin	|	Function	|
|-------|-----------------------|
|PA8	|	LED array 1	|
|PA11	|	LED array 2	|
|PA12	|	LED array 3	|
|PA15	|	LED array 4	|
|PC13	|	Debug LED	|


### MPU6050 Connection
|Pin	|	Function		|
|-------|-------------------------------|
|PB8	|	SCL(Software Simulated)	|
|PB9	|	SDA(Software Simulated)	|
|PB5	|	EXTI9_5(MPU6050_INT)	|

### Debug
|Pin	|	Function	|
|-------|-----------------------|
|PA9	|	USART1_TX	|
|PA10	|	USART1_RX	|
|PA13	|	SWDIO		|
|PA14	|	SWCLK		|
